# 音频相关概念
## 音频格式

  **PCM (Pulse Code Modulation)**
  -未压缩的原始音频数据格式，是音视频开发中最常用的基础格式。
  
  **MP3**
  -有损压缩，兼容性好，应用广泛。
  
  **AAC**：
  -有损压缩，效率比 MP3 更高，音视频开发中常用。
    
  **WAV**：
  -基于 PCM 的容器格式，通常为无损，文件体积较大。
  
## 音频参数

  **采样率 (Sample Rate)**
  -每秒对声音信号的采样次数，单位 Hz。
  -采样率越高，声音还原度越好，但数据量也越大。
  
  **位深 (Bit Depth)**
  -每个采样点用多少位二进制数表示，决定了声音的动态范围。
  -常见值：8bit、16bit、24bit、32bit。
  -位深越高，声音细节和动态范围越丰富。
  
  **声道数 (Channels)**
  -声音信号的通道数量。
  -常见类型：
    - 单声道 (Mono)：1 个声道
    - 双声道 (Stereo)：2 个声道（左+右）
    - 多声道：4个声道以上
    
  **码率 (Bitrate)**
  -单位时间内传输的音频数据量，单位 kbps。
  -码率越高，音质越好，文件体积也越大。
  -码率 = 采样率 × 位深 × 声道数

# 音频简易开发流程

1.  rkmedia系统初始化
2.  设置AI通道属性（音频采集）
3.  使能AI通道
4.  设置AO通道属性（音频播放）
5.  使能AO通道
6.  绑定AI通道与 AO 通道
7.  阻塞程序（等待退出）
8.  解除通道绑定
9.  失能AO通道
10. 失能AI通道

# 代码示例（音频环回测试）
```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include "rkmedia_api.h"
#include <stdio.h>
#include <assert.h>
#include <fcntl.h>
#include <pthread.h>
#include <signal.h>
#include <stdbool.h>
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <time.h>
#include <unistd.h>
#include "rkmedia_api.h"

int main()
{
    //rkmedia系统初始化
    RK_MPI_SYS_Init();
    //设置ai通道
    AI_CHN_ATTR_S ai_attr;
    ai_attr.pcAudioNode = "default";
    ai_attr.enSampleFormat =RK_SAMPLE_FMT_S16;
    ai_attr.u32Channels =2;
    ai_attr.u32SampleRate =16000;
    ai_attr.u32NbSamples =1024;
    ai_attr.enAiLayout =AI_LAYOUT_NORMAL;
    RK_MPI_AI_SetChnAttr(0, &ai_attr);

    //使能ai通道
    RK_MPI_AI_EnableChn(0);

    //设置ao通道
    AO_CHN_ATTR_S ao_attr;
    ao_attr.pcAudioNode = "plughw:1,0";
    ao_attr.enSampleFormat =RK_SAMPLE_FMT_S16;
    ao_attr.u32Channels = 2;
    ao_attr.u32SampleRate = 16000;
    ao_attr.u32NbSamples = 1024;
    RK_MPI_AO_SetChnAttr(0, &ao_attr);

    //使能ao通道
    RK_MPI_AO_EnableChn(0);

    //绑定接口
    MPP_CHN_S src_chn,des_chn;
    src_chn.enModId = RK_ID_AI;
    src_chn.s32DevId = 0;
    src_chn.s32ChnId = 0;
    des_chn.enModId = RK_ID_AO;
    des_chn.s32DevId = 0;
    des_chn.s32ChnId = 0;
    RK_MPI_SYS_Bind(&src_chn,&des_chn);
    
    printf("输入任意键退出\n");
    getchar();

    //解除绑定
    RK_MPI_SYS_UnBind(&src_chn,&des_chn);
    //失能ao
    RK_MPI_AO_DisableChn(0);
    //失能ai
    RK_MPI_AI_DisableChn(0);
    return 0;
}
